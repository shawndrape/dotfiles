# Dotfiles

## Dependencies

Install GNU Stow
```
brew install stow
```

## Reminders

* `.gitconfig` - Unless brew is installing apps automatically during 
a setup, be sure to have installed the GitHub CLI app and done the ssh
auth. Otherwise, connection will not work
